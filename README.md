The **Uber Trip Analysis Dashboard** is an interactive Business Intelligence project built using **Power BI** to analyze Uber trip data and uncover insights into booking trends, trip distances, revenue, vehicle performance, and customer travel behavior.

The dashboard enables stakeholders to monitor business performance through dynamic visualizations, KPIs, and filters, helping support data-driven operational decisions.

---

# 🎯 Project Objectives

- Analyze Uber trip booking patterns.
- Track revenue and booking performance.
- Identify peak travel periods.
- Understand trip distance distribution.
- Compare vehicle category performance.
- Monitor business KPIs through interactive dashboards.

📊 Dashboard Pages

### 1️⃣ Overview Analysis

Provides a high-level summary of Uber operations including:

- Total Bookings
- Total Revenue
- Average Trip Distance
- Average Trip Time
- Vehicle Category Distribution
- Booking Trends
- Interactive KPI Cards
- Dynamic Filters

2️⃣ Time Analysis

Analyzes trip demand over time.

Key insights include:

- Daily Trends
- Weekly Trends
- Monthly Booking Trends
- Peak Hours Analysis
- Trip Volume by Time
- Seasonal Demand Patterns

---

### 3️⃣ Details

Detailed trip-level information including:

- Individual Trip Records
- Pickup & Drop-off Details
- Fare Information
- Distance Covered
- Trip Duration
- Vehicle Category
- Interactive Data Table
